# WDAC Testing Checklists

This directory contains comprehensive checklists for testing WDAC policies on different environments:

- Windows 10/11 (AD and non-AD)
- Windows Server (AD and non-AD)

Each checklist provides step-by-step instructions for:
- Environment preparation
- Policy deployment
- Testing procedures
- Validation criteria
- Troubleshooting steps

These checklists enable anyone to clone the repository and immediately begin testing WDAC policies on their systems.
